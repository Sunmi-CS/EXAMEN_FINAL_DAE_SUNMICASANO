# AnimeGrid - SPA (Examen Final - SUNMI CASAÑO SUAREZ)

## Tecnologías
- React + Vite
- Zustand (estado global)
- React Router
- Bootstrap
- API: Jikan (https://api.jikan.moe/)

## Instalación
```bash
cd examen-final-scasano-dae
npm install
npm run dev
````
